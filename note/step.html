# Express 建立專案網頁框架

# 1. 建立專案資料夾並以VSCode開啟

# 2. 在資料夾上點選右鍵/在整合式終端機中開啟

# 3. 初始化專案

執行

npm init -y

# 完成後資料夾內會新增：

  • packsge.json

# 4. 安裝express-generator套件

windows系統執行

npm install express-generator

Mac系統執行

sudo npm install express-generator

Mac系統需要開管理員權限安裝，因此會要求輸入電腦密碼

# 完成後資料夾內會新增：

  • node_modules 資料夾
  • packsge-lock.json

# 5. 使用express建立框架

執行

express --view=ejs

確認輸入 y

# 完成後資料夾內會新增：

  • bin 資料夾
  • public 資料夾
  • routes 資料夾
  • views 資料夾
  • app.js

# 6. 安裝預設套件

執行

npm install

套件會被安裝在node_modules 資料夾中

# 7. 打開專案預設頁面

執行

npm start

用瀏覽器打開 localhost:3000

預設port為3000，可從app.js中修改

# 8. 完成
